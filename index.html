HoneyTrap is an open-source framework for honeypots.

Using HoneyTrap you can create sensor-, low-, medium- and high interaction honeypots. Combine low- interaction services together with proxies to create a interesting environment.

Central management

Using HoneyTrap Agents you can deploy many agents across your organization and connect them to a central HoneyTrap server.

Easy service development

Adding new services is as easy as implementing one method. All the rest is being done by the HoneyTrap framework.

Interactive Dashboard

Use the interactive dashboard to monitor your HoneyTrap deployment live.

Advanced logging system

Using the advanced logging system you'll be able to log to all kind of destinations, like Elasticsearch, Splunk, Raven Server, Console, Slack and Kafka.
